Features

The beta version of Evrimo is equipped with the following features.

Multiple output display options

Optimizing Vehicle Route

The optimized vehicle route can be viewed on different devices and softwares. It can be viewed on Google Earth through a KML file, on Sygic through ITF file, or as simple text file

Gps Based Navigation

The KML file shows the detailed street paths on Google Earth desktop application. The ITF file can be downloaded on a mobile phone with Sygic for detailed GPS based navigation.

Automatic Genrated Files

The text file lists down the sequence of stops in their visit sequence along with total distance in Kilometers. All these files are automatically generated and can also be downloaded on your device for future reference.

Multiple Input Options

Inputs

The inputs to Evrimo include path details such as city name, path names and geocodes of locations and warehouse locations in a particular path, and other miscellaneous details.

No Upload Limit

The user can give inputs to Evrimo through form based interface or can upload bulk data through CSV files

Evrimo Data Base

The bulk upload option can be used to easily import existing data into Evrimo database.

Maintain History

Individual Database Management

Maintains database for each user.

Privacy And Security

All the inputs, output files, profile information and billing related details are maintained in secure database servers.

Improvement

Security improve everyday

Paths with numerous destination

Optimization Of Path

Each user can optimize vehicle paths in any city of the world that are supported by Evrimo

Upto 50 Destination

For a standard user, up to 50 destinations can be included in a single path.

Customization Request

On customized request, the number of destinations can be increased up to 100 per path.

Customization For Number of Destination

Moreover, the number of destinations can also be increased beyond 100, but at the expense of increased computation delay.
